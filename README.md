# AWS Cloud Practitioner Pass プライバシーポリシー

AWS Cloud Practitioner Pass は、AWS Certified Cloud Practitioner (CLF-C02) の学習向け非公式アプリです。本アプリは、学習履歴、設定、プレミアム購入状態、Firebase Analytics による利用状況イベントを、学習体験と機能品質の改善のために取り扱います。

本アプリは、問題本文、選択肢本文、解説本文、自由入力、氏名、メールアドレス、Apple の決済識別子の生値、広告IDを Firebase Analytics に送信しません。

無料ユーザー向けにバナー広告およびインタースティシャル広告を表示する場合があります。

本アプリはAmazon Web Services, Inc.またはその関連会社が提供、承認、推薦、監修する公式アプリではありません。
